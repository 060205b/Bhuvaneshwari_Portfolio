Traffic Accident Severity Prediction

Project Overview

The Traffic Accident Severity Prediction System leverages machine learning techniques to predict the severity of traffic accidents based on various parameters. This project employs a Random Forest Classifier, enhanced through GridSearchCV for optimal hyperparameter tuning. A web application built using Flask provides a user-friendly interface for inputting parameters and receiving predictions on accident severity.

Problem Statement

Traffic accidents are a major public safety concern, leading to numerous injuries and fatalities each year. Accurately predicting the severity of these accidents is crucial for emergency response planning and traffic management. The challenge lies in developing a robust model that can analyze various influencing factors—such as driver age, weather conditions, and collision types—to provide reliable severity predictions.

Model Building

The modeling process involved:

  • Data Preprocessing: The dataset was cleaned and split into training and testing sets (X_train, X_test, y_train, y_test) for evaluation.
  • Model Selection: A Random Forest Classifier was chosen for its effectiveness in handling complex datasets while minimizing overfitting.
  • Hyperparameter Tuning with GridSearchCV: GridSearchCV was utilized to fine-tune hyperparameters such as:
    • Number of estimators: The number of trees in the forest.
    • Max depth: Limits the maximum depth of trees to mitigate overfitting.
    • Minimum samples split: Specifies the minimum number of samples needed to split an internal node.

This rigorous tuning process ensures that the model performs at its best, achieving higher accuracy in predictions.

Model Saving

Upon completion of training and tuning, the best-performing model was saved using Pickle. This allows for easy deployment and usage in future applications without the need for retraining.

Solution

The result of this project is a robust predictive model capable of classifying traffic accident severity into three categories: Low, Medium, and High. This model provides valuable insights that can assist drivers and traffic authorities in making informed decisions, ultimately contributing to improved road safety and effective resource allocation in emergency situations.

Features

  • Machine Learning Prediction: A finely tuned Random Forest model that predicts traffic accident severity (Low, Medium, High).
  • Web Interface: Developed using Flask, enabling users to interact with the model through a simple web form.
  • Feature Importances: Visualized insights into the features that significantly affect accident severity predictions.
  • SHAP and LIME Plots: Integrated tools for model interpretation, providing clarity on how individual features influence prediction outcomes.
