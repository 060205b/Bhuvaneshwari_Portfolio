Restaurant Ratings Analysis Project

Problem Statement

In a competitive restaurant industry, understanding what drives customer satisfaction is crucial for success. Many restaurant owners struggle to gauge how various factors—such as pricing, delivery options, and cuisine types—affect their ratings. How can we identify key influencers of restaurant ratings and predict future ratings based on existing data?

Solution Overview

This project aims to address the problem by leveraging data analysis and machine learning techniques to uncover insights into the factors influencing restaurant ratings and to build predictive models. The analysis focuses on:

  • Data Exploration
  • Feature Engineering
  • Model Development
  • Data Visualization

1. Data Exploration and Preprocessing

Data Loading and Quality Check: Loaded the dataset, assessed its structure, and checked for missing values, ensuring a clean dataset for analysis.

Categorical Encoding: Converted categorical variables into numeric format for model compatibility, facilitating more effective analysis.

Data Filtering: Focused on restaurants offering both table booking and online delivery to understand the impact of these features on customer ratings.

2. Feature Engineering and Selection

Key Features: Identified critical features for modeling, such as Average Cost for two, Price range, Votes, and booking options. The target variable, Aggregate rating, served as the output we aimed to predict.

Scaling and Normalization: Applied standardization to features using StandardScaler, ensuring consistent scaling and enhancing model performance.

Train-Test Split: Divided the data into training and testing sets (80-20 split) to evaluate model accuracy effectively.

3. Model Development and Evaluation

Linear Regression: Developed a baseline model to predict restaurant ratings and conducted statistical analysis using OLS regression to understand feature significance and relationships.

OLS Regression Results

Visualization of Actual vs. Predicted Values: This plot helps in visually assessing the performance of the Linear Regression model, highlighting how closely the predicted ratings align with actual ratings.

Decision Tree Regressor: Implemented a Decision Tree model to capture non-linear patterns in the data.

Random Forest Regressor: Applied an ensemble method, the Random Forest Regressor, to improve prediction accuracy by averaging multiple decision trees.

Model Comparison: Compiled a comparison of model performances, summarizing key metrics (MAE and R²) to identify the most effective model for predicting restaurant ratings.

4. Data Visualization and Insights

Top 20 Cuisines and Aggregate Ratings: This bar plot visualizes the average ratings of the top 20 cuisines, revealing customer preferences in a clear and interpretable manner.

Distribution Analysis: Utilized histograms, boxplots, and violin plots to examine the distribution of ratings.

Ratings by Top Cities: Analyzed how aggregate ratings vary across cities. This violin plot illustrates the distribution of ratings across cities, providing insights into regional differences in restaurant performance.

Feature Interactions: Created pair plots to visualize the relationships between key features and the target variable.
